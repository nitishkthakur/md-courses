# Dimensionality Reduction Masterclass

This course covers the full spectrum of dimensionality reduction — from classical linear methods like PCA and LDA to deep nonlinear approaches like autoencoders and VAEs. Each file is self-contained with theory, intuition, and practical guidance on when to use each method. The course is structured so you can read linearly or jump directly to the technique that fits your problem.

## Course Files

| File | Description |
|------|-------------|
| [overview.md](overview.md) | Algorithm taxonomy, workhorse vs advanced split, decision table, evaluation guide |
| [pca.md](pca.md) | PCA + Randomized/Incremental/Kernel/Sparse PCA — the universal starting point for linear DR |
| [lda.md](lda.md) | Linear Discriminant Analysis — supervised linear projection maximizing class separability |
| [nmf.md](nmf.md) | NMF — parts-based non-negative decomposition for topic modeling and spectral analysis |
| [ica.md](ica.md) | ICA — independent component extraction for signal separation and artifact removal |
| [tsne.md](tsne.md) | t-SNE — gold-standard nonlinear visualization; local structure, perplexity tuning |
| [umap.md](umap.md) | UMAP — faster than t-SNE, preserves global structure, supports transform() for new data |
| [manifold-methods.md](manifold-methods.md) | Isomap, LLE, Spectral Embedding, MDS, PHATE — topology-preserving methods |
| [clustering-as-dr.md](clustering-as-dr.md) | Clustering as DR — prototype distances, GMM posteriors, Bag of Visual Words |
| [autoencoder-dr.md](autoencoder-dr.md) | Autoencoder + VAE — deep nonlinear DR with generative capability |
| [random-projections.md](random-projections.md) | Random Projections — Johnson-Lindenstrauss, LSH, streaming-scale DR |

For the full course map, algorithm decision guide, and evaluation framework, start with [overview.md](overview.md).
